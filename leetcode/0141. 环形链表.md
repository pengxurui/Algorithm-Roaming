## [141. 环形链表](https://leetcode.cn/problems/linked-list-cycle/)

## 题目描述

给你一个链表的头节点 head ，判断链表中是否有环。

## 题目考点

快慢指针

## 题解
 
```
class Solution {
    fun hasCycle(head: ListNode?): Boolean {
        var slow = head
        var fast = head
        while (null != fast && null != fast.next) {
            slow = slow!!.next
            fast = fast.next!!.next
            if (slow == fast) {
                return true
            }
        }
        return false
    }
}
```

**复杂度分析：**

- 时间复杂度：O(n)
- 空间复杂度：O(1)
