## [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

## 题目描述

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

## 题目考点

队列、双指针、哈希表

## 题解一（队列）
 
```
class Solution {
    fun lengthOfLongestSubstring(s: String): Int {
        if (s.length <= 1) {
            return s.length
        }
        val queue = LinkedList<Char>()
        val selected = HashSet<Char>()
        var maxLength = 0
        for (everyElement in s) {
            if (selected.contains(everyElement)) {
                // 剔除重复字符
                while (true) {
                    val node = queue.poll();
                    selected.remove(node)
                    if (node == everyElement) {
                        break;
                    }
                }
            }
            queue.offer(everyElement)
            maxLength = Math.max(maxLength, queue.size)
            selected.add(everyElement)
        }
        return maxLength
    }
}
```

**复杂度分析：**

- 时间复杂度：O(n)
- 空间复杂度：O(n) 

## 题解二（队列）

```
class Solution {
    fun lengthOfLongestSubstring(s: String): Int {
        if (s.length <= 1) {
            return s.length
        }
        var left = 0
        var maxLength = 0
        val selected = HashSet<Char>()
        for (index in s.indices) {
            val curNode = s[index]
            if (selected.contains(curNode)) {
                while (true) {
                    var node = s[left++]
                    selected.remove(node)
                    if (node == curNode) {
                        break;
                    }
                }
            }
            selected.add(curNode)
            maxLength = Math.max(maxLength, index - left + 1)
        }
        return maxLength
    }
}
```


**复杂度分析：**

- 时间复杂度：O(n)
- 空间复杂度：O(M)，M 是字符集大小
