## [206. 反转链表](https://leetcode.cn/problems/reverse-linked-list/)

## 题目描述

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

## 题目考点

链表

## 题解
 
```
class Solution {
    fun reverseList(head: ListNode?): ListNode? {
        if(null == head || null == head.next){
            return head
        }
        val prefix = reverseList(head.next)
        head.next.next = head
        head.next = null
        return prefix
    }
}
```

**复杂度分析：**

- 时间复杂度：O(n)
- 空间复杂度：O(1)
