## [509. 斐波那契数](https://leetcode.cn/problems/fibonacci-number/)

## 题目描述

斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。

## 题目考点

动态规划

## 题解（动态规划）
 
```
class Solution {
    fun fib(n: Int): Int {
        if (n < 2) {
            return n
        }

        var first = 0
        var second = 1
        for (index in 2..n) {
            val temp = first
            first = second
            second = temp + first
        }
        return second
    }
}
```

**复杂度分析：**

- 时间复杂度：O(n)
- 空间复杂度：O(1)
